## 🎮 About

This project is a recreation of the classic **Pong game**, designed to demonstrate core programming and game development concepts in a simple and beginner-friendly way.

---

## ✨ Features

- 🏓 Classic Pong gameplay  
- 🎯 Smooth paddle controls  
- ⚡ Ball movement with collision detection  
- 📊 Score tracking system  
- 🧠 Clean and readable code  

---

## 🛠️ Tech Stack

- Python

---

## 📦 Installation

```bash
git clone https://github.com/hosama-adem/pong-game-hos.git
cd pong-game-hos
pip install -r requirements.txt
```

---

## ▶️ Run the Game

```bash
python main.py
```
